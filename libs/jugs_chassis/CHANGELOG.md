# Changelog

All notable changes to this project will be documented in this file.

## [0.1.2] - 2026-02-27

### Changed

- Added configurable test attributes (for example, latency) to improve test coverage and observability.
- Reduced logging granularity from per-building output and expanded log handling capacity to lower log volume while supporting higher throughput.

## [0.1.1] - 2026-02-14

### Changed

- Improved README installation instructions.

## [0.1.0] - 2026-02-13

### Added

- Initial release.
- Logging configuration derived from `jug_lca_buildings`.
