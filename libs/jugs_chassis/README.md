# jugs-chassis is deprecated

`jugs-chassis` is deprecated and has been replaced by
[`sabu-chassis`](https://pypi.org/project/sabu-chassis/).

## What to use instead

Install the replacement package:

```bash
pip install sabu-chassis
```

Project page:

- https://pypi.org/project/sabu-chassis/

Source:

- https://github.com/demianAdli/sabu/tree/main/libs/sabu_chassis

## Why

The package was renamed from `jugs-chassis` to `sabu-chassis` as part of
the JUGS to Sabu project rename.

## Status

No new development should target `jugs-chassis`. Use `sabu-chassis` for all
new installs and dependency declarations.
